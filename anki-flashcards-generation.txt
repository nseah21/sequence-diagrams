end

opt refine generation
Student->Frontend:Enter additional instructions
end

  Frontend->Backend: Validate parameters and \n     forward to backend
  activate Backend

    Backend->MongoDB: Fetch flashcards marked \n   as inaccurate by user
    activate MongoDB

    MongoDB-->Backend:
    deactivate MongoDB

    Backend->AI Server: Send parameters and inaccurate\n        flashcards to AI Server
    activate AI Server

      alt user uploaded a file
      AI Server->AI Server:Use vector store to generate tuples of \n     (question, answer, description)
      activate AI Server
      
      AI Server-->AI Server:
      deactivate AI Server
      
      else else
      AI Server->AI Server:Use base LLM to generate tuples of \n     (question, answer, description)
      activate AI Server
      
      AI Server-->AI Server:
      deactivate AI Server
      
      end

    AI Server-->Backend:
    deactivate AI Server

  Backend-->Frontend:
  deactivate Backend

  Frontend->Backend: Save flashcards
  activate Backend

    Backend->MongoDB: Persist flashcards to DB
    activate MongoDB

    MongoDB-->Backend:
    deactivate MongoDB

  Backend-->Frontend:
  deactivate Backend

Frontend-->Student:
deactivate Frontend
